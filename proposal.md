# Object Guesser

## Repository
https://github.com/M1CHAELjr/final_project.git

## Description
The object guesser would have a list of items, and based off of the answers given from the user will change the outcome of what object is guessed.  But practically this could be used as a potential job finder or even as a way to find intrests or artistic styles. 

## Features
- Feature 1: Item list
  - Will be the list of items that the computer will have to choose from
- Feature 2: Terminal questions
  - The user will be asked within the terminal about the object they choose, in yes or no questions
- Feature 3: Custom responses
  - Will answer differently based on wheter or not you answer yes or no to a question. Meaning every single responce will warrant a unique prompt.
- Feature 4: Answer
  - If the user answered honestely the computer should guess the correct object.

## Challenges 
- Creating a list long enough and with enough items that will make it interesting
- Making sure the code stays along the given path and doesn't stray, based off user input
- Balancing items that can be guessed within the given number of questions
